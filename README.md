# Traffic Tool Essentials

A traffic light phase editor for Cities: Skylines II.

## Features

- **Phase Editor** - Create custom traffic light phases for any intersection
- **Copy & Paste** - Copy phases between intersections
- **Sequential Mode** - Traffic lights cycle through phases in order
- **Green Wave** - Synchronize multiple intersections for coordinated green lights
- **Split Phasing** - Separate opposing traffic for safer intersections
- **Pedestrian Phases** - Add exclusive pedestrian crossing phases
- **Manual Control** - Take direct control of any traffic light
- **Statistics** - Monitor timer, priority, vehicle flow, and lane occupancy
- **Configurable Hotkeys** - Set up keyboard shortcuts for quick access

- Visual Studio 2022
- .NET SDK
- Cities: Skylines II with modding tools

### Build Steps
1. Clone this repository
2. Open `TrafficLightsEnhancement.sln` in Visual Studio
3. Restore NuGet packages
4. Build solution
